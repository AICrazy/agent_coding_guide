# ACCEPTANCE_EVIDENCE (验收验证证据模板)

Optional advanced artifact: use this only when a project wants a standalone acceptance evidence file in addition to the default `acceptance_report.md`. (高级可选产物：仅在项目需要把验收证据从默认验收报告中单独拆分时使用)

## Scope (范围)
- Covered `PR-*` items. (覆盖的产品需求)

## Executed Validation Cases (执行的验收验证)
- `TC-A-*`: objective, method, result, evidence. (目标、方法、结果、证据)

## Traceability Notes (追踪说明)
- Related verification and design refs. (关联验证和设计引用)

## Defects and Risks (缺陷与风险)
- Related `DEF-*` refs or `none`. (关联缺陷)
- Remaining validation risks. (剩余验收风险)

## Conclusion (结论)
- `pass|fail|partial`. (结论)
