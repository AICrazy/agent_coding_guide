# SYSTEM_DESIGN (系统设计模板)

## Goal (目标)
- Design scope and baseline target. (设计范围与基线目标)

## System Boundary (系统边界)
- Context, major actors, and external dependencies. (上下文、主要参与者和外部依赖)

## Subsystems and Interfaces (子系统与接口)
- `SD-*`: subsystem or interface item. (子系统或接口项)
- Mapped `SYS-*|SWR-*`. (对应需求)

## Runtime View (运行视图)
- Main runtime flow, deployment/runtime assumptions, and control boundaries. (主运行流、运行假设、控制边界)

## NFR Allocation (非功能分配)
- Performance/security/usability allocation. (非功能分配)

## Risks (风险)
- Architecture risks and mitigations. (架构风险与缓解措施)
