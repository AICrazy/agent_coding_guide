# SECURITY_ASSESSMENT (安全评估模板)

## Scope (范围)
- Sensitive change present: `yes|no`. (是否涉及敏感变更)
- Scope summary. (范围说明)

## Trust Boundaries (信任边界)
- External calls, credentials, secrets, or private data. (外部调用、凭据、密钥或私有数据)

## Controls (控制措施)
- Existing controls. (现有控制)
- Added controls. (新增控制)

## Verification (验证)
- Security checks or evidence refs. (安全检查或证据)

## Conclusion (结论)
- `pass|fail|not-applicable`. (结论)
