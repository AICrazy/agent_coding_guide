# SYSTEM_TEST_REPORT (系统测试报告模板)

Optional advanced artifact: use this only when a project chooses split per-level verification reports instead of the default consolidated `verification_report.md`. (高级可选产物：仅在项目选择拆分分层验证报告时使用)

## Scope (范围)
- Covered end-to-end system behaviors and NFRs. (覆盖端到端系统行为与非功能需求)

## Executed Test Cases (执行用例)
- `TC-S-*`: objective, method, result, evidence. (目标、方法、结果、证据)

## Environment Result (环境结果)
- Runtime/deployment/config assumptions validated. (运行时、部署和配置假设验证)

## NFR Result (非功能结果)
- Usability, layout, performance, security, compatibility as applicable. (按需记录)

## Defects and Security (缺陷与安全)
- `DEF-*` refs. (缺陷引用)
- `docs/quality/security_assessment.md` ref or `not-applicable`. (安全评估)

## Conclusion (结论)
- `pass|fail|partial`. (结论)
