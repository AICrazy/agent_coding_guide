# DEFECT (缺陷)

- `id`: defect id (缺陷编号)
- `title`: short title (短标题)
- `severity`: `sev1|sev2|sev3` (严重级别)
- `priority`: `p0|p1|p2` (优先级)
- `phase_found`: phase found (发现阶段)
- `repro`: repro steps (复现步骤)
- `actual`: actual result (实际结果)
- `expected`: expected result (预期结果)
- `impact`: impact scope (影响范围)
- `root_cause`: root cause (根因)
- `fix`: fix summary (修复摘要)
- `verification`: verification result (验证结果)
- `owner`: owner (负责人)
- `status`: current status (当前状态)
