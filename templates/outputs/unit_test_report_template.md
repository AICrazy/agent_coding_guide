# UNIT_TEST_REPORT (单元测试报告模板)

Optional advanced artifact: use this only when a project chooses split per-level verification reports instead of the default consolidated `verification_report.md`. (高级可选产物：仅在项目选择拆分分层验证报告时使用)

## Scope (范围)
- Covered `SWR-*|DD-*`. (覆盖的软件需求或详细设计项)

## Executed Test Cases (执行用例)
- `TC-U-*`: objective, method, result, evidence. (目标、方法、结果、证据)

## Coverage Notes (覆盖说明)
- Covered items. (已覆盖项)
- Gaps and rationale. (缺口与原因)

## Defects (缺陷)
- Related `DEF-*` refs or `none`. (关联缺陷或 `none`)

## Conclusion (结论)
- `pass|fail|partial`. (结论)
