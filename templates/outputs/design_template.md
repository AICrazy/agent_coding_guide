# DESIGN (设计文档模板)

## Goal (目标)
- Design target for current scope. (当前范围的设计目标)

## Architecture (架构)
- System boundary. (系统边界)
- Main modules. (主要模块)
- Key responsibilities. (关键职责)

## State and Data Flow (状态与数据流)
- Core state model. (核心状态模型)
- Main transitions. (主要状态流转)
- Data ownership. (数据归属)

## Decisions (设计决策)
- Key technical decisions. (关键技术决策)
- Rejected options if relevant. (必要时记录放弃方案)

## Code Target (代码落点)
- Chosen code target. (选定代码落点)
- Reuse rule or layout note. (复用规则或布局说明)

## Constraints (约束)
- Build/runtime constraints. (构建或运行时约束)

## Risks (风险)
- Design risks. (设计风险)
- Follow-up notes. (后续说明)
