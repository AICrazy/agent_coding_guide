# TEST_REPORT (测试报告模板)

## Scope (范围)
- Tested scope. (测试范围)

## Checks (检查项)
- Automated checks. (自动检查)
- Manual checks. (手动检查)

## Acceptance Criteria Coverage (验收标准覆盖)
- Criterion. (验收标准)
- Verification method. (验证方法)
- Result: `pass|fail|unverified`. (结果)
- Evidence. (证据)

## Viewport Matrix (视口矩阵，仅 Web 适用)
- Viewport size. (视口尺寸)
- Checks: vertical scroll, horizontal overflow, overlap/clipping, core UI visibility/usability. (检查项)
- Result: `pass|fail|unverified`. (结果)
- Evidence. (证据)

## Results (结果)
- Passed checks. (通过项)
- Failed checks. (失败项)

## Defect Refs (缺陷引用)
- `docs/defects/<id>.md` for each blocker or waived defect. (每个阻塞/豁免缺陷的引用)

## Security Ref (安全引用)
- `docs/security.md` if written. (如存在则引用 `docs/security.md`)

## Evidence (证据)
- Command outputs, logs, or notes. (命令输出、日志或说明)

## Conclusion (结论)
- Test status. (测试结论)
- Fallback role if rework is required. (若需返工，记录回退角色)
