# ACCEPTANCE (验收文档模板)

## Scope Review (范围复核)
- Scope in complete. (范围内已完成)
- Scope out unchanged. (范围外未扩展)

## Gate Review (门禁复核)
- Required checks passed. (必要检查已通过)
- No unresolved blocker. (无未解决阻塞项)

## Acceptance Criteria Gate (验收标准门禁)
- Each required acceptance criterion has evidence and a final result. (每条必需验收标准均有证据和最终结果)
- Unverified required criteria. (未验证的必需验收标准)

## Evidence (证据)
- Requirements reference. (需求引用)
- Design reference. (设计引用)
- Test evidence: `docs/test_report.md` if generated; else inline summary here. (测试证据：若生成测试报告则引用，否则在此内联摘要)
- Defect refs if any. (如有则引用缺陷记录)
- Security ref if any. (如有则引用安全记录)

## Decision (结论)
- `accept` or `reject`. (通过或拒绝)

## Risks (风险)
- Remaining risks. (剩余风险)

## Retro (复盘)
- Short lessons learned. (简短复盘)
