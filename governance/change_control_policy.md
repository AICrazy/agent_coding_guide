# CHANGE_CONTROL_POLICY

Canonical runtime rules live in `governance/workflow_protocol.md`.

After `G1`, any changed required behavior needs `CR-*`.

Each `CR-*` records:
- affected `PR-*|SYS-*|SWR-*`
- affected `SD-*|DD-*|artifacts`
- impacted verification levels
- whether acceptance revalidation is required
- baseline/version decision
- closure evidence refs
