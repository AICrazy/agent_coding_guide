# Snake Game Demo (贪吃蛇示例)

## Overview (项目概览)
- A browser-based Snake game with a simple single-page experience. (一个浏览器中的单页贪吃蛇游戏)
- Built for quick play, clear controls, and stable core gameplay. (强调易上手、操作清晰、核心玩法稳定)

## Features (功能特性)
- Start, pause/resume, restart, and game-over flow. (开始、暂停/继续、重开、结束流程)
- Keyboard-based movement control. (键盘方向控制)
- Manual difficulty controls that change the snake speed. (支持手动难度调节并改变蛇移动速度)
- Food, score, snake growth, wall collision, and self collision. (食物、分数、成长、撞墙、自撞)
- Clean status display for score and running state. (分数和运行状态展示)
- Responsive layout for desktop, tablet, and mobile, with desktop single-screen-first presentation. (适配桌面、平板、手机，并优先保证桌面端主界面单屏展示)

## Controls (操作方式)
- Arrow keys move the snake. (方向键控制移动)
- Optional `WASD` support may be enabled in implementation. (实现中可选支持 `WASD`)
- Pause/resume and restart are exposed in the game UI. (暂停/继续与重开通过界面提供)

## Product Scope (产品范围)
- Single-page browser game. (单页浏览器游戏)
- Lightweight browser-native implementation. (轻量原生浏览器实现)
- Fixed-grid board with selectable speed levels. (固定网格棋盘与可选速度档位)
- Responsive UI that keeps the main gameplay area readable across common viewports. (响应式界面，确保常见视口下主游戏区域可读可用)
- No account system, backend service, leaderboard, or multiplayer mode. (不包含账号、后端、排行榜、多人模式)

## Audience (目标用户)
- Casual players who want a quick browser game. (希望快速体验浏览器小游戏的用户)
- Demo and learning scenarios that need a compact web example. (需要紧凑 Web 示例的演示或学习场景)
